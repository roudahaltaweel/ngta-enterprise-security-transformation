# NGTA Enterprise Security Transformation Program (ESTP)

## Overview

The **NGTA Enterprise Security Transformation Program (ESTP)** is a long-term cybersecurity portfolio project developed for a fictional UAE federal government authority called the **National Government Technology Authority (NGTA)**.

This project simulates a real enterprise environment where cybersecurity is continuously designed, implemented, and improved using industry best practices, university knowledge, and professional certifications.

Instead of creating multiple unrelated cybersecurity projects, this repository represents one evolving enterprise that grows throughout my Bachelor's degree in IT Security & Forensics and future cybersecurity certifications.
---

## Project Objectives

The objectives of this project are to:

- Design a realistic enterprise cybersecurity environment.
- Apply cybersecurity concepts learned through university coursework and professional certifications.
- Develop practical experience in enterprise security architecture and risk management.
- Build professional documentation that reflects industry best practices.
- Demonstrate continuous learning by expanding the project throughout my academic and professional journey.
  ---

## Scope

This project covers multiple cybersecurity domains, including:

- Enterprise Architecture
- Network Security
- Identity and Access Management (IAM)
- Security Operations (SOC)
- Incident Response
- Digital Forensics
- Risk Management
- Governance, Risk & Compliance (GRC)
- Cloud Security
- Security Policies and Standards
